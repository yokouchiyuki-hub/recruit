★ Design Rules
デザインルール
このプロジェクト全体で守るべきデザイン上の決まりごとを記録します。
RULE 01 フォントサイズ最小値 — 14px

サイト内で使用するすべてのフォントサイズは 14px を最小値 とする。
ラベル・注釈・メタ情報など小さく見せたい要素も 14px 未満は使用しない。

✓ OK  font-size: 14px / 16px / 18px / 20px …
✗ NG  font-size: 13px / 12px / 11px / 10px / 9px …
例外： em単位で指定する装飾テキスト（例：アイコン内の相対サイズ）は対象外。
RULE 02 画像ホバー時のscale拡大禁止

写真・カード内画像へのカーソルオン時に transform: scale() による拡大モーション を使用しない。

✗ NG  img:hover { transform: scale(1.05); }
✓ OK  brightness / opacity などの filter 変化は可
RULE 03 カードホバー時の位置移動禁止

カードホバー時に translateY・rotate などの位置移動アニメーション を使用しない。

✗ NG  .card:hover { transform: translateY(-4px); }
✓ OK  box-shadow / border-color などの視覚的変化は可
